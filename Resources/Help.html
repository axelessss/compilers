Компилятор

 Компилятор - это приложение для редактирования текстовых файлов с возможностью компиляции и выполнения программ на языке программирования. Приложение разработано как часть курсовой работы по теме "Разработка пользовательского интерфейса (GUI) для языкового процессора" с использованием C# и .NET Framework 4.8.

Функциональности

    1.Создание, открытие и сохранение текстовых файлов в различных форматах (.txt, .cs, все файлы).
    2.Возможность редактирования текста с поддержкой форматирования.
    3.Отображение номеров строк.
    4.Возможность компиляции и выполнения программ на языке программирования.

Панель инструментов:

  • Файл
  • Правка
  • Правка
  • Текст
  • Пуск
  • Справка

Файл

  • Создать
    Создает новый файл для редактирования.
  • Открыть
    Открывает существующий файл для редактирования.
  • Сохранить
    Сохраняет текущее состояние файла.
  • Сохранить как
    Сохраняет текущее состояние файла с новым именем.
  • Выход
    Закрывает текстовый редактор.

Правка

  • Отменить
    Отменяет последнее действие.
  • Повторить
    Повторяет последнее отмененное действие.
  • Вырезать
    Удаляет выбранный текст и копирует его в буфер обмена.
  • Копировать
    Копирует выбранный текст в буфер обмена.
  • Вставить
    Вставляет текст из буфера обмена в текущее место курсора.
  • Удалить
    Удаляет выбранный текст.
  • Выделить все
    Выделяет весь текст в редакторе.

Текст

  • Постановка задачи
    Открывает html-документ с описанием задачи.
  • Грамматика
    Открывает html-документ с описанием грамматики.
  • Классификация грамматики
    Открывает html-документ с классификацией грамматики по Хомскому.
  • Метод анализа
    Открывает html-документ с описанием метода синтаксического анализа.
  • Диагностика и нейтрализация ошибок
    Открывает html-документ с описанием метода диагностики и нейтрализации ошибок.
  • Тестовый пример
    Открывает в окне редактора файл с готовым тестовым примером.
  • Список литературы
    Открывает html-документ со списком полезной литературы по теме работы.
  • Исходный код программы
    Ссылка на репозиторий GitHub с исходным кодом программы.

Пуск (F5)

Сканирование текста и запуск синтаксического анализатора с выводом всех ошибок в таблицу во вкладке "Парсер" и всех найденных лексем в таблицу во вкладке "Сканер".

Справка

  • Вызов справки
    Отображает окно справки с описанием функций редактора.
  • О программе
    Показывает информацию о текстовом редакторе и его версии.
